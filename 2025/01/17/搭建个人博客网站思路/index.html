部署 Hexo 的两种方法

第一种：本地部署 Hexo

Hexo 是一个轻量级的博客框架，提供多种主题风格，可以省去我们很多建站时的脑力劳动，提高工作效率。本地部署 Hexo 相对简便，主要环境需求是 Git 和 Node.js。可以通过 Hexo 官方文档按步骤进行搭建。

  • Hexo 官方文档

第二种：将本地 Hexo 部署到云服务器

这种方法是本文的重点，主要讲解如何将本地部署的 Hexo 部署到云服务器，实现通过公网域名访问。

服务器购买

可以选择华为云、腾讯云、阿里云等云服务商来购买云服务器，很多服务商都提供新人专享优惠。购买完服务器后，我们还需要进行一些配置。

配置安全组

购买服务器后，记得配置安全组以开放必要的端口，这样才能进行访问。常见的端口包括：

  • SSH: 22
  • Tomcat: 80
  • Nginx: 80
  • Hadoop: 9870

例如，使用华为云的 CentOS 7 镜像时，我们需要特别注意开放这些端口，确保服务可以正常运行并且能够通过公网访问。
