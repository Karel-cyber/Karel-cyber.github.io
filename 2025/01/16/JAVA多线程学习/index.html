JAVA多线程学习

关键字（thread）

概念

  • 线程是进程的最基本的单位，是CPU调度的最小单位，离不开CPU。

  • 单进程多线程，多核CPU，能够提高资源的使用效率。

  • 在同一时间需要完成多项任务的时候需要线程

创建线程的三种方式

  • 继承Thread类 ——– 单继承

  • 实现Runnable接口 ————— 无返回值任务

  • 实现Callable接口 —————— 有返回值任务

    以上Runnable，Callable不具有线程的启动的能力

    Thread类只可以单继承，不能多继承其他类

两种实现办法的区别在于：

  • 一个是继承Thread类，一个是实现了runnable接口
  • 代码实现逻辑不一样，Thread类具备这个线程启动，runnable不具备
  • Thread类局限性较大，runnable可以较为灵活，能够实现热拔插效果

实现callable接口就不写了。
